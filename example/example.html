# Simple

* Write _markdown_ inside a single HTML template
* Open the template in your browser
* Done

--

    <!DOCTYPE html>
    <html>
      <head>
        <meta name="viewport"
          content="width=device-width, initial-scale=1">
        <link rel="stylesheet" type="text/css"
          href="../dist/basic.css">
      </head>
      <body>

    <pre class="slides">
    # Markdown here!
    </pre>
        <script src="../dist/bundle.js"></script>
      </body>
    </html>
